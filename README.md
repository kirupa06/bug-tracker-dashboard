# 🐞 Bug Tracking Dashboard  
A **simple bug tracking dashboard** built using **React, Redux Toolkit, and TypeScript**. It helps track, manage, and resolve issues efficiently.  

🚀 **Live Demo**: [Bug Tracking Dashboard](https://bug-tracker-dashboard-kirupa06s-projects.vercel.app/)  

![Bug Tracking Dashboard](https://github.com/user-attachments/assets/75071b3e-e6d2-42ef-9a6f-a4a743ad02ee)  

---

## ✨ Features  
✅ File and categorize bugs  
✅ Track bug status (**Open, In Progress, Resolved, Closed**)  
✅ Manage issue resolutions effectively  
✅ User-friendly and intuitive UI  
✅ **Redux Toolkit** for state management  
✅ **TypeScript** for better scalability  
✅ **Vercel Deployment** for instant access  

---

## 🛠️ Technologies Used  
- ⚛️ **React** – Frontend framework  
- 🔄 **Redux Toolkit** – State management  
- 🏗 **TypeScript** – Type-safe development  
- 🎨 **Styled Components** – Styling  
- 🚀 **Vercel** – Deployment  

---

## 📦 Installation & Setup  
Follow these steps to clone and run the project locally:  

```bash
# Clone the repository  
git clone https://github.com/your-username/bug-tracker-dashboard.git  

# Navigate to the project folder  
cd bug-tracker-dashboard  

# Install dependencies  
npm install  

# Start the development server  
npm run dev
